# About
This week's work covers building basic web pages using HTML and CSS. There are two pages in this project — a general webpage with some HTML elements, and a cat profile page styled with an external CSS file.
## Files 
| File           | Description                                      |
|----------------|--------------------------------------------------|
| `webpage.html` | A simple HTML page with basic elements           |
| `cat.html`     | A profile page for "Pochi the Cat"               |
| `cat.css`      | External stylesheet linked to `cat.html`         |

## webpage.html
A basic HTML page that demonstrates commonly used HTML elements:
- Paragraph text using `<p>`
- A heading with `<h1>`
- Bold text using `<b>`
- An anchor tag linking to Google
- An unordered list `<ul>` with fruit items (Apple, Mango, Banana, Grapes)
## cat.html
A profile page for a fictional cat character named **Pochi the Cat**. It includes:
- A centered main heading `<h1>` with the cat's name
- An introduction paragraph about the character
- A cat image embedded using base64 data, floated to the right via CSS
- A profile section using an ordered list `<ol>`
- A links section with two anchor tags pointing to an external reference
- A horizontal rule `<hr>` and a centered "Last Updated" note at the bottom
## cat.css
A short external stylesheet used by `cat.html`:
- The cat image is floated to the **right** so text wraps around it
- `<h2>` headings are styled in **blueviolet**
- The `<h1>` heading is styled in **red**
## How to View
Just open either `.html` file in any browser — no setup or server needed.
- Open `webpage.html` for the basic HTML demo
- Open `cat.html` for the styled cat profile page (make sure `cat.css` is in the same folder)
