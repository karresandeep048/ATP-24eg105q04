# Week 6 – Full Stack & React Practice Project
## Overview
This repository contains a collection of practice applications developed during Week 6 as part of MERN Stack and React learning activities. The projects focus on frontend development using React and Vite, along with backend integration using Node.js, Express.js, and MongoDB.

The folder includes:
1. MERN Employee Management Application
2. Users Management React Application
3. Counter Application using React Context API

---

# Features
- CRUD operations
- React component-based architecture
- REST API integration
- MongoDB connectivity
- Context API state management
- Responsive UI design

# Technologies Used
- React.js
- Vite
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios
- Tailwind CSS

---

# Installation

## Frontend
```bash
npm install
npm run dev
```

## Backend
```bash
npm install
npm start
```
