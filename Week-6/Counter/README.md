# Counter App

A React Context API demo with 4 synchronized counter panels sharing a single global state.

## Getting Started

```bash
npm install
npm run dev
```

## Features
- Global state via React Context API
- 4 panels all synced to one counter value
- Increment / Decrement / Reset per panel
- Global +10 / −10 / Reset All controls
- Color feedback: green for positive, red for negative
- Responsive layout

## Stack
- React 19 + Vite
- Tailwind CSS v4
- DM Mono + Syne fonts
