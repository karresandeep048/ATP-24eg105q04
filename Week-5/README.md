# About
This folder contains all the HTML and CSS files I made during my web development lab sessions. Each file is a different concept or exercise we covered in class.
### Bootstrap1 (Boorstrap1.css + HTML)
A basic webpage layout with a hero section, three-column card layout, and a footer. Used simple custom CSS to replicate a Bootstrap-style layout — hero banner with a heading, paragraph, and a primary button, followed by three cards with secondary buttons. Footer shows the college name centered.
### Bootstrap2 (Bootstrap2.css + HTML)
A political-themed webpage layout called "Libertarians" — made it for practice with flexbox layouts and responsive design. Has a red hero section with text and an emoji icon, and three dark red info cards below. Includes media queries for mobile responsiveness (stacks cards vertically under 768px).
### FlexBox (FlexBox.css + HTML)
A card layout page with a top navbar and four styled blog-style cards using flexbox. Each card has a circular image pulled from Unsplash, a colored category tag (Travel, Tech, Fashion), a heading, and a short description. Learned how to use `flex-wrap` and `box-shadow` here.
### pic (pic.css + HTML)
A styled user table with a dark background and a teal container. Shows a list of users with their profile picture (circular), name, email, phone, and a comment column. Practiced styling tables with `border-radius` and `overflow: hidden` on the table element.
### tstyle (tstyle.css + HTML)
A simple HTML table with `colspan` to merge header cells across columns — shows a "Title", two "Subtitles", data rows, and "Subtotal" rows at the bottom. Mainly used to understand how `colspan` works in HTML tables.
### stylet (stylet.css + HTML)
A grade table using both `colspan` and `rowspan`. Shows Biology, Chemistry, and Physics subjects for two students (John Doe and Miriam Luther), with Practical and Theory rows merged using `rowspan="2"`. Main exercise for understanding how rowspan and colspan work together.
## Concepts Practiced
- Flexbox layouts and responsive design
- HTML tables with colspan and rowspan
- CSS box shadows, border-radius, and custom styling
- Media queries for mobile responsiveness
- Linking external CSS stylesheets
- Using Unsplash images and icon CDNs

- VS Code
