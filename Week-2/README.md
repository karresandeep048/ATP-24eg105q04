
## Overview
This folder contains different JavaScript programs that I practiced to understand basic and intermediate concepts. The programs include real-world examples like shopping cart, employee data, student records, etc.
## Topics Covered
* Array methods (filter, map, reduce, find, findIndex)
* Objects and arrays
* Functions and rest parameters
* Classes and basic OOP
* Shallow copy and deep copy
* Promises and asynchronous operations
* setTimeout and setInterval
* Simple module-based structure (import/export)
## Programs Included
* Movie filtering and rating calculation
* Employee payroll processing
* Shopping cart system
* Student performance system
* Bank transaction analyzer
* Temperature and marks analysis
* E-commerce mini project (cart, payment, discount)
* Task manager with validation
* Library book management system
* OTP timer and async examples
## How to Run
1. Install Node.js
2. Open terminal in project folder
3. Run:
   node filename.js
